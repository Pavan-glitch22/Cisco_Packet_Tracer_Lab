Project Overview

This repository contains my hands-on lab work using Cisco Packet Tracer as part of the Cisco NetAcad curriculum.

The labs cover fundamental to advanced networking concepts including:

LAN setup
Wireless networking
CLI configuration
SDN (Software Defined Networking)
IoT automation
