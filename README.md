## 🔬 Lab Summaries

### Lab 01 — Create a Simple Network
- **Objective:** Build a basic LAN by connecting PCs to a switch and verifying ICMP connectivity.
- **Topology:** 2 PCs → Cisco 2960 Switch (FastEthernet)
- **Key Concepts:** OSI Layer 2, IP addressing, ICMP, straight-through cables

---

### Lab 02 — Create a Simple Network 2
- **Objective:** Extend the basic network with more end devices across a single switch.
- **Topology:** Multiple PCs → Cisco 2960 Switch (star topology)
- **Key Concepts:** MAC address table, switch forwarding, broadcast domain, IP subnetting

---

### Lab 03 — Connect Devices using Wireless Technologies
- **Objective:** Configure a WLAN using a home router/AP and connect wireless clients.
- **Topology:** Linksys WRT300N → Laptops (wireless)
- **Key Concepts:** WLAN, SSID, WPA2, DHCP, 802.11 standards

---

### Lab 04 — Create Realistic Structured Cabling in the Physical Workspace
- **Objective:** Simulate enterprise-grade structured cabling using the Physical Workspace view.
- **Topology:** Server rack with switch, patch panel, and wall ports in a building layout
- **Key Concepts:** Structured cabling, patch panels, IDF/MDF, TIA/EIA standards

---

### Lab 05 — Edit Topologies
- **Objective:** Modify an existing topology by adding/removing devices and updating configurations.
- **Topology:** Pre-built topology extended with new switches and PCs
- **Key Concepts:** Topology modification, network scalability, traceroute

---

### Lab 06 — Explore Device Configuration Using the CLI (Console)
- **Objective:** Access and configure a Cisco router/switch via CLI through console connection.
- **Topology:** PC → Cisco Router (rollover/console cable)
- **Key Concepts:** Cisco IOS modes, console access, interface config, `show` commands

---

### Lab 07 — Manage and Configure your Network using a Network Controller
- **Objective:** Use an SDN-based controller to centrally manage and push configs to devices.
- **Topology:** Network Controller → Multiple Managed Switches
- **Key Concepts:** SDN, centralized management, VLAN policies, northbound/southbound APIs

---

### Lab 08 — Modify an Existing Script for an IoT Thing
- **Objective:** Edit an existing Python/JS automation script to change IoT device behavior.
- **Topology:** IoT devices → Home Gateway (scripted automation)
- **Key Concepts:** Event-driven programming, IoT scripting, sensor/actuator logic

---

### Lab 09 — Create Your Own IoT Thing
- **Objective:** Build a custom IoT device from scratch with sensors, actuators, and scripts.
- **Topology:** Custom IoT Device → Home Gateway → IoT Server
- **Key Concepts:** Custom IoT development, Python scripting, device registration, smart home automation

---

### Lab 10 — Monitor Your Network Using a Network Controller
- **Objective:** Use the controller's monitoring dashboard to observe real-time network health.
- **Topology:** Network Controller monitoring multiple switches, routers, and end devices
- **Key Concepts:** Network monitoring, SDN dashboard, real-time analytics, fault detection
